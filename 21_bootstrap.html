        웹 개발 편의성을 위해 만들어진 프론트엔드 프레임워크로써 
        html, css, javascript 로 구성 되어있다.

        또한 드롭다운, 버튼, 네비게이션 바, 폼, 모달(팝업), 등의 
        UI 기본 틀을 미리 정의하여 개발자들이 쉽게 사용할 수 있도록 만들어두었다.
