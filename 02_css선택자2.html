CSS 선택자 내부 속성 방식

기본 속성 선택자

기본 선택자 뒤에 [] 기호를 사용해서 속성과 속성 값을 사용할 수 있다. 주로 id 나 class 속성이 아닌 것들에 적용
선택자[속성명=속성값]{설정 할 값}

반응 선택자

사용자의 행동에 따라 달라지는 선택자

        사용자가 클릭을 한 태그 -> 선택자:active{설정 할 값}
        사용자의 마우스가 올라온 태그 -> 선택자:hover{설정 할 값}
