CSS 기본 선택자

선택자(selector) 란? 특정한 html 요소(태그)를 선택할 때 사용하는 기능 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다. css : 이쁘게 꾸미는 것 Java Script : 기능을 부여하는 것

모든 선택자

HTML 문서 안의 모든 태그를 선택할 때 전체 선택자 사용 가능

태그 선택자

아이디 선택자

HTML 문서 내에 해당하는 ID 속성을 가진 태그를 선택한다.

아이디를 나타내는 기호 (#) -> #아이디명{꾸밀 값}
